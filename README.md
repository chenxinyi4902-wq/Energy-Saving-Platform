# Ecopulse - Energy Saving Platform

## Project Status

* The final testing version of the Energy Saving Platform has now been completed.
* The official release version of the Energy Saving Platform has now been completed.

## Current Features

* User login and logout
* Personal energy-saving data dashboard
* Monthly energy-saving target and progress tracking
* Automatic points settlement
* Energy usage history search
* Platform-wide energy-saving leaderboard
* Reward redemption with points
* Redemption history records
